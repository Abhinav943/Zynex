# Security Policy

## Supported Versions

Zynex takes the security of its validation engine very seriously. We currently apply security patches to the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Zynex, please do NOT open a public issue.

Instead, please send an email directly to the maintainer at **abhinav09prajapati@gmail.com**. We will treat your report with the utmost priority and respond within 48 hours to discuss the timeline for a patch and a coordinated public disclosure.
